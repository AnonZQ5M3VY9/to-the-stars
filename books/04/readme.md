虫洞（英语：wormhole），又称 爱因斯坦-罗森桥（英语： Einstein—Rosen bridge ），是宇宙中可能存在的连接两个不同时空的狭窄隧道。

**[良子](https://tts.determinismsucks.net/wiki/Ryouko)的力量和[章鱼](https://tts.determinismsucks.net/wiki/Cephalopod)的虫洞技术的主题可能会很突出。**
